
  # Obaid Salem

  This is a code bundle for Obaid Salem. The original project is available at https://www.figma.com/design/CBPjyMIwonFsIpfk9eERwr/Obaid-Salem.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Contact form (Appwrite)

  Add these variables to your `.env` file:

  - `VITE_APPWRITE_CONTACT_URL` (required) — your Appwrite Function HTTP endpoint
  - `VITE_APPWRITE_API_KEY` (optional) — only if your function requires `x-appwrite-key`
